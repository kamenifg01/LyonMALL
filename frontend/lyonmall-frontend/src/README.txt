Intégration de AWS Simple Notification Service (SNS)
1. Configurer AWS SNS :
  1.1. Créer un compte AWS :

  Si tu n'as pas encore de compte AWS, crée-en un sur AWS.
  
  1.2. Créer un topic SNS :
  Accède à la console AWS SNS.
  Clique sur "Create topic" pour créer un nouveau topic SNS.
  Note l'ARN (Amazon Resource Name) du topic pour l'utiliser plus tard.
  
  1.3. Configurer les autorisations IAM :
  Assure-toi que le rôle ou l'utilisateur IAM que tu utilises a les permissions nécessaires pour publier des messages sur le topic SNS.
